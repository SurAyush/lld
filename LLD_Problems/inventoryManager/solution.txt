We need to build an inventory manager
Products --> warehouses --> inventory
inventory -> single instance as a manager --- singleton design pattern

Each Product represents a chunk og same items (100 packets of Lays)

Products will have different ways to replenish, -- strategy Design pattern  -- Bulk & JustInTime
Product will belong to different categories --> extended class

Observer Design Pattern to notify observers regarding low stock alerts.

We can use Builder Desing Pattern to build complex product.
